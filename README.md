# An automated bidding bot developed for the 2026 Playtech Internship assignment 
## Overview
**Video Score**: Uses a rating calculated based on the video's attributes which is therefore utilized for bidding. Ceil for passing is held higher to avoid videos that wouldn't benefit the bet. <br>
**Aggressiveness**: Starts off boldy, but gradually becomes more reserved in terms of bet amounts. <br>
**Win ratio**: Win ratio is calculated based on the bets won from the last 100 videos, low ratio results in a higher bet.
